Supplementary Information

About

Online-Fix.Me is a trusted source of pirated games. It uses the game SpaceWar in Steam to gain access to its multiplayer capabilities. You don't actually need to install it, just add it in your Steam library. Don't believe us? Take it from the pirating community or directly from r/PiratedGames Megathread.

Installation

Specific instruction will be given at every piracy page. The one given here is the common general instructions.

  1. You usually need to extract GAMENAME.VERSION-OFME.rar. Extraction password is online-fix.me
  2. You will then need to run the game's executable (i.e., GAMENAME.exe) to play.

Multiplayer

The online fix is usually already in the game files when you extract the primary .rar. However, if problems occur with the online fix (i.e., multiplayer won't work), extract this file: GAMENAME_Fix_Repair_Steam_V3_Generic.rar located in the Fix Repair folder that usually comes with the torrent download. It has the same extraction password: online-fix.me Copy the all the files back to the game files. Confirm overwrite of files.

Language

Sometimes, the game will be set to Russian. Here is how to change it:

  1. In this directory: GAMENAME/binaries/x64, open OnlineFix.ini in a notepad.
  2. Change Language=russian to Language=english.
  3. Save and Exit.

Note however, sometimes the GAMENAME might not exactly be the game's name. So just look around the folders where the binaries folder might be nested.
