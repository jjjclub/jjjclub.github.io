Instructions

  1. Extract both .rar files simultaneously. Extraction password is online-fix.me
  2. Run stateofdecay2.exe to play.

  3. To play multiplayer, go to in-game Setting > Multiplayer and check all the following:

    • Host Multiplayer Game
    • Open to friends.
  4. Create a campaign and finish the tutorial. In the menu, select Mutliplayer to see available friends and invite them.

Notes

  1. If the game refuses to launch, shorten the directory where you placed the game files.
  2. By default, the game will be in russian. Check this page to see how to change it.
  3. Steam must be open to play singleplayer and Xbox app must be open for multiplayer.
  4. A Microsoft/Xbox account is needed to play multiplayer. The initial screen will ask you to log in after you launch the game.
  5. You must complete the tutorial before inviting friends. The tutorial ends when the player has cured the other survivor you were with during the tutorial.
