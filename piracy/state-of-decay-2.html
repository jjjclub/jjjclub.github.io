Instructions

  1. Extract both .rar files simultaneously.
  2. In the Fix Repair folder, extract the SoD2JE_Fix_Repair_Steam_V4_Generic .rar file, copy the contents and paste it in the game files. Extraction password is online-fix.me. Confirm overwrite of files.
  3. Run stateofdecay2.exe to play.

  4. To play multiplayer, go to in game Setting > Multiplayer and check all the following:

    • Host Multiplayer Game
    • Open to friends.
  5. Create a campaign and finish the tutorial. In the menu, select Mutliplayer to see available friends and invite them.

By default, the game will be set to Russian. Here is how to change it:

  1. In this directory: State of Decay 2 Juggernaut Edition\StateOfDecay2\Binaries\Win64, open OnlineFix.ini.
  2. Change Language=russian to Language=english.
  3. Save and Exit.

Notes

  1. If the game refuses to launch, shorten the directory where you placed the game files.
  2. Steam must be open to play singleplayer and Xbox app must be open for multiplayer.
  3. A Microsoft/Xbox account is needed to play multiplayer. The initial screen will ask you to log in after you launch the game.
  4. You must complete the tutorial before inviting friends. The tutorial ends when the player has cured the other survivor you were with during the tutorial.
