Instructions

Initial Setup

  1. You only need to do this section once.
  2. Download and extract all files to their own respective folders. Extraction password is cs.rin.ru
  3. Close Steam and it must remain closed from this point onward.
  4. Open CreamAPI_Release_v4.5.0.0_Hotfix folder, open log_build folder, copy all files, paste to auto-creamapi-2.1.7 folder.
  5. In cauto-creamapi-2.1.7 folder, run auto-creamapi.exe.
  6. If you do not have it yet, you will be automatically prompted to download .Net Core. After installing it, run auto-creamapi.exe again.
  7. AutoCreamAPI has now been setup.

General Usage

  1. Keep Steam closed. You need to do this section everytime you want to unlock something.
  2. Run auto-creamapi.exe, click "Select DLL file" icon in the top right.
  3. Go to the game directory of your selected game, select the steam_api64.dll and click open.
  4. Ensure that the following are done:
    • Set to english (if not already set)
    • Check "Try to bypass game-specfic protection"
    • Check "Unlock all DLCs (if possbile)"
  5. Click "Get DLCs for AppID", then wait.
  6. Click save in the bottom right. If it says "Saving Succesful" in the lower left, then you may close the application.

Game Specific Usage

Company Heroes 2 Factions

  1. Keep Steam closed.
  2. Download and extract .rar. Extraction password is cs.rin.ru.
  3. Copy all the files inside, then paste to the CoH2 Steam directory.
  4. Run COH2 in Steam to play.

Notes

  1. You must own a game in Steam to use CreamAPI. It only unlocks DLCs, not the actual game itself.
